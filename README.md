# Social Robotics

Welcome to the **Social Robotics** repository!  
Here you’ll find **guides** and **example code** to help you learn how to use each of the robots available for your course project.

---

## Robot Access

You can use the robots during **lab sessions**, but they’re also available **24/7** in **room XX**.  

To ensure everyone gets fair access, we use a **Google Calendar booking system**.  
This allows you to:

- **Book time slots** for the robot you need  
- **See when robots are already in use**  [Calendar](https://calendar.google.com/calendar/embed?src=c_35c092cef07756df8d191b45ba92cb2c31aad46752e61553c695ea96d1a9cb69%40group.calendar.google.com&ctz=Europe%2FLisbon)
- **Cancel your booking** if you can’t attend  
- **Check which group is using each robot**

Please be respectful and responsible:
- Cancel your booking if you’re not going to show up — this helps others use the robots.
- Communicate with other groups to coordinate usage fairly.
